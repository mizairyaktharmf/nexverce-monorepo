5. Best Practices

üìù Writing Content

DO:
‚úÖ Use clear, simple language
‚úÖ Break text into short paragraphs (3-4 lines max)
‚úÖ Use headings to organize content
‚úÖ Add images every 300-400 words
‚úÖ Include FAQs at the end
DON'T:
‚ùå Write walls of text
‚ùå Skip headings
‚ùå Forget to add images
‚ùå Use too many different fonts/colors
‚ùå Leave blocks empty

üé® Design Tips

Element Best Practice
Headings Use H2 for main sections, H3 for subsections
Images Use high-quality images, compress before upload
Spacing Add spacer blocks between major sections
Colors Stick to brand colors (purple gradient)
Buttons Use clear, action-oriented text

‚ö° Speed & SEO

  • Optimize Images: Compress images before uploading (use TinyPNG.com)
  • Alt Text: Always add descriptive alt text to images
  • Reading Time: Fill in the reading time field (helps SEO)
  • Author Name: Add your name (builds trust)
  • Meta Description: Use the AI SEO generator for blog posts

6. Common Mistakes to Avoid

‚ùå Mistake #1: Too Much Text, No Images

Problem: Long paragraphs without visual breaks make content hard to read.
Solution: Add an image block every 300-400 words. Use spacers and dividers.

‚ùå Mistake #2: Forgetting to Fill Author Name

Problem: Content looks less trustworthy without author attribution.
Solution: Always fill in "Author Name" field before publishing.

‚ùå Mistake #3: Not Using Headings

Problem: Content is hard to scan and bad for SEO.
Solution: Use Heading blocks (H2, H3) to organize your content into clear sections.

‚ùå Mistake #4: Empty Blocks

Problem: Leaving empty blocks creates ugly gaps.
Solution: Delete any blocks you're not using. Click the üóëÔ∏è button.

‚ùå Mistake #5: Too Many CTAs

Problem: Too many buttons overwhelm readers.
Solution: Use 1-2 CTA buttons maximum per article.

‚ùå Mistake #6: Not Saving Drafts

Problem: Losing all your work if browser crashes.
Solution: Click "üíæ Save Draft" every 10-15 minutes while writing.

8. Troubleshooting

üîß Common Issues & Solutions

Problem: Block won't delete

Solution: Refresh the page and try again. Make sure you're clicking the üóëÔ∏è button, not the duplicate button.

Problem: Can't drag blocks

Solution: Make sure you're grabbing the grip handle (‚ãÆ‚ãÆ) on the left side of the block.

Problem: Image won't upload

Solution:

  • Check file size (max 5MB recommended)
  • Use JPG or PNG format
  • Make sure you have internet connection
  • Try compressing the image first

Problem: Content not saving

Solution:

  • Check your internet connection
  • Make sure you're logged in
  • Try clicking "Save Draft" again
  • Copy your content to a text file as backup

Problem: Preview looks different from live site

Solution: This is normal. The preview is simplified. Always check the published page to see final result.

Problem: Can't find a specific block

Solution: Use the search box at the top of the left sidebar to find blocks by name.
